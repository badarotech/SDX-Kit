# studies/

Cada rodada de pesquisa é um "lote" numa subpasta aqui, com uma pasta `raw/` para os
arquivos originais e um `sources.md` (o manifesto das fontes). Para começar:
- rode `/sources knowledge/research/studies/<seu-lote>` para catalogar um lote inteiro, ou
- rode `/add-source <caminho-ou-URL>` para adicionar uma fonte por vez.
